# Personal Portfolio Website

## 📌 Project Description
This project is a responsive personal portfolio website built using HTML, CSS, and JavaScript. It serves as the foundation of my professional portfolio and showcases my background, projects, and contact information.

The website highlights my interests in Artificial Intelligence, Machine Learning, and Web Development, while demonstrating front-end development skills and basic data handling using APIs.

---

## 🚀 Features
- Responsive design (desktop, tablet, mobile)
- Hero section with background image and profile picture
- About Me section with interactive hover text effect
- Projects section with:
  - Hover animations
  - Live search/filter functionality
- Contact form with validation and confirmation message
- Dark/Light theme toggle
- AI News section:
  - Fetches real-time AI-related news using a public API
  - Displays dynamic content
  - Handles loading and error states

---

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Responsive Design)
- JavaScript (DOM manipulation, Fetch API)
- Public API (Hacker News API)
